## 목표

마크다운 문서를 sqlite-vec(vec0) + 로컬 MiniLM(384d) 임베딩으로 인덱싱해 **codegraph.db 단일 파일**에 통합하고, codegraph MCP 커맨드 output에 편입. 코드 그래프 무회귀.

## 확정 결정 (사용자 승인)

1. **DB**: codegraph.db 단일 파일 통합 (별도 mdast.db 아님). vec0는 확장 로드 후 런타임 생성(스키마 무조건 X).
2. **의존성**: @xenova/transformers = optionalDependencies, lazy. sqlite-vec는 **이미 dependencies에 존재**. 옵트인 = `CODEGRAPH_DOCS=1` env / `--with-docs`. 미설치 시 graceful skip.
3. **출력**: 신규 codegraph_docs + context(1차) + node/impact(governing 문서 있을 때만 게이트). search/callers/callees/trace 제외.
