사용자와의 모든 소통은 반드시 한글(한국어)로 한다. 응답 본문, AskUserQuestion의 질문·옵션 라벨·설명·프리뷰, 승인/결재 요청을 전부 한국어로 작성한다.

**Why:** 2026-06-18 EXECUTE 진입 시 영어로 작성한 Decision Gate 질문(AskUserQuestion)을 사용자가 거부하며 "한글로 다시 물어봐. 앞으로 소통은 반드시 한글로 하도록해"라고 명시 지시함.

**How to apply:** 사용자 대면 텍스트는 전부 한국어. 코드·심볼명·파일경로·기술 식별자(hex, fileID, BLK 코드 등)는 원문 유지 가능. [[feedback_approval_requests]] 승인 요청도 한국어 + 옵션 비교/근거 형식 유지.
